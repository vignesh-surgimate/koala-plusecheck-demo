Drop-in CI Compatibility

GitHub Actions

Ships in-repo: smoke.yml on PRs, regression.yml nightly with sharding and Allure publishing. CI=true auto-tunes retries (2) and workers (4).

Jenkins

Drop-in compatible: a declarative stage calling npm run test:matrix with env credentials binding — same npm script contract, no framework changes needed.
