The Sweep — 7 × 4 with evidence

One command logs in as every clinical role at every profile and attaches login + dashboard screenshots to the report — instant visual regression evidence for leadership.

// Viewports come from the single device matrix — add a profile, it joins the sweep.
for (const viewport of viewportMatrix) {
  for (const role of VALIDATED_ROLES) {
    test(`[${viewport.name}] ${role} login @smoke`, async ({ page }, testInfo) => {
      await page.setViewportSize({ width: viewport.width, height: viewport.height });
      await loginPage.loginAndWaitForHome(user.username, user.password);
      await testInfo.attach(`${role}-${viewport.name}-dashboard`, { path: shot, contentType: 'image/png' });
    });
  }
}
npm run test:viewports   # full 7 × 4 sweep with screenshot evidence
