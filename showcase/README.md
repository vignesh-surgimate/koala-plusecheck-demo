# Surgimate Framework — Enterprise Documentation Portal

Static dark-violet documentation portal (GitHub Pages ready, no backend).

## Deploy to GitHub Pages
Option A — gh-pages branch: push the CONTENTS of this folder to a `gh-pages`
branch root and enable Pages → "Deploy from branch" → gh-pages / root.

Option B — docs folder: copy this folder's contents into `/docs` on your main
branch and enable Pages → main / docs.

Option C — preview locally: just open `index.html` in a browser
(everything works over file:// except Google Fonts fallback).
